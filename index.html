Proyectos

DeportLand

El proyecto consiste en el desarrollo del front-end de un prototipo de e-commerce para una tienda específica, con el objetivo de demostrar las ventajas de una plataforma digital en términos de diseño y usabilidad. Aunque no se implementó la funcionalidad de ventas en línea, el prototipo muestra la estructura y el flujo visual de un sitio de e-commerce, destacando su interfaz intuitiva, la organización de productos y la facilidad de navegación. Este prototipo busca evidenciar cómo un e-commerce bien diseñado puede mejorar la experiencia de compra y atraer a más usuarios.

Tecnologías utilizadas: Java, JavaScript, HTML, CSS.

Mictlan

Mictlan es un proyecto de e-commerce para un bar con temática prehispánica, cuyo producto estrella es la cerveza artesanal Quijada. El objetivo es ampliar el mercado de esta cerveza, permitiendo su venta en línea. El proyecto incluirá el desarrollo tanto del front-end como del back-end, ofreciendo una plataforma fácil de usar, segura y optimizada, que permita a los clientes comprar Quijada y otros productos relacionados, mientras se expande la presencia digital del bar.

Tecnologías utilizadas: Java, Spring Boot, JavaScript, HTML, CSS, MySQL.
